9. Interactive Visualization with Tableau

Topics and Notes
  1. Register an account with Tableau Public and install Tableau Public Desktop on your machine.
  2. Basic Statistical Charts in Tableau
    • a. Bar chart, pie chart, donut chart
    • b. Histogram and density curves
    • c. Time series plot (line plot)
    • d. Scatter plot and buble plot
    • e. Maps and tree maps
  3. Derived variables and Tableau built-in functions
  4. Filters
  5. Tableau working data set: Superstore Data
  6. A Brief Note on Basic Statistical Charts with Tableau HTML RMD SOURCE
Assignments
Recreate the presidential election map using Tableau.
  1. Data sets: Presidential Election Data FIPS to Geocode
  2. Data management and visual creation tasks:
    • a. Extract on 2020 presidential election data: year
    • b. Only include Democrats and Republican votes: party
    • c. Include variables: state_po, county_name, county_fips, party, candidatevotes
    • d. Merge the above data with FIPS to Geocode Datausing the FIPS as the primary key.

    • Hint: You can simply load the data set you created in the last assignment to Tableau and create a Tableau map that satisfies the following conditions.

    • e. Create an interactive choropleth map to display the presidential election results at the county level using two different colors to represent the two parties. After finishing the Tableau map, you save it to the Tableau Public Server (need to sign in to your account) and submit the URL of your map on the Tableau server to the D2L drop box.
    Assignment Due: Sunday, 11:30 PM
